    Glossary

    A link

    Action Ability link

    An ability indicated by the word “Action” that a player can use as their action for the turn. See Abilities and Effects

    Active Player link

    The player currently taking an action. See Turn Structure

    Ambush link

    A keyword that allows a unit to ready and attack when played. See Keywords (Related: Ready)

    Arena link

    A shared zone where units are played. There are two arenas: ground and space. See Game Zones (Related: Ground Arena, Space Arena)

    Aspect link

    Colored icons representing different philosophies (Vigilance, Command, Aggression, Cunning, Villainy, Heroism). See Game Concepts

    Aspect Penalty link

    Additional cost when playing cards with aspects not provided by your leader or base. See Game Concepts

    Attach link

    To connect an upgrade to a unit. See Card Types (Related: Upgrade)

    B link

    Base link

    A card type representing a location; each player has one. See Card Types

    Base Zone link

    The zone where a player’s base and leader (when not deployed) are located. See Game Zones

    Bounty link

    A keyword ability that allows an opponent to resolve an effect when the card is defeated or captured. See Keywords

    Brick link

    (Slang) A hand with no playable cards or a deck that consistently draws unplayable combinations.

    C link

    Combat Damage link

    Damage dealt during the second step of an attack. See Player Actions

    Constant Ability link

    An ability that is always active while the card is in play. See Abilities and Effects

    Controller link

    The player who currently controls a card in play. See Game Concepts

    Copy link

    A card with identical printed attributes to another card. See Game Concepts

    Cost link

    The number of resources required to play a card or use an ability. See Game Concepts

    Curve link

    (Slang) The distribution of card costs in a deck, often referring to a smooth progression from low to high costs.

    D link

    Damage link

    Represented by damage counters on cards with HP. See Game Concepts

    Deck link

    A player’s collection of cards used to play the game. See Game Zones

    Defeat link

    When a card leaves play due to damage or an ability. See Game Concepts

    Discard link

    To move a card to its owner’s discard pile. See Game Zones

    E link

    Event link

    A card type with a one-time effect when played. See Card Types

    Exhaust link

    To turn a card sideways, indicating it has been used. See Game Concepts (Related: Ready)

    F link

    Friendly link

    Cards a player controls. See Game Concepts

    Fish link

    (Slang) To draw cards hoping to find a specific answer or combo piece.

    G link

    Grit link

    A keyword that gives a unit +1 power for each damage on it. See Keywords

    Ground Arena link

    The zone where ground units are played. See Game Zones (Related: Arena, Space Arena)

    Goldfish link

    (Slang) To play out your strategy without interaction, as if playing against an opponent doing nothing.

    H link

    Hand link

    The zone where a player holds cards not yet played. See Game Zones

    HP (Hit Points) link

    How much damage a card can take before being defeated. See Card Types

    Hate Card link

    (Slang) A card specifically included to counter a popular strategy or deck type.

    I link

    Initiative link

    Determines the first player each round, represented by the initiative counter. See Player Actions

    J link

    Jank link

    (Slang) Unconventional or seemingly suboptimal deck builds or strategies.

    K link

    Keyword link

    A bolded ability with specific rules, such as Ambush or Grit. See Keywords

    L link

    Leader link

    A double-sided card type representing a character, with different abilities on each side. See Card Types

    Lasting Effect link

    An effect that persists for a specified duration. See Abilities and Effects

    M link

    Meta link

    (Slang) The current state of popular strategies and deck types in competitive play.

    Mirror Match link

    (Slang) A game where both players are using very similar or identical deck strategies.

    N link

    Nested Ability link

    An ability triggered during the resolution of another ability. See Abilities and Effects

    O link

    Overwhelm link

    A keyword that allows excess damage to be dealt to the opponent’s base. See Keywords

    Owner link

    The player whose deck a card started in. See Game Concepts

    P link

    Play link

    To use a card from hand by paying its cost and putting it into play or resolving its ability. See Player Actions

    Power link

    How much damage a unit deals in combat. See Card Types

    Ping link

    (Slang) To deal a small amount of damage, often to remove a Shield token or defeat a low-HP unit.

    R link

    Raid link

    A keyword that gives a unit bonus power when attacking. See Keywords

    Ready link

    An upright card that hasn’t been used this turn. See Game Concepts (Related: Exhaust)

    Resource link

    A card in the resource zone used to pay costs. See Game Concepts

    Restore link

    A keyword that heals damage from your base when the unit attacks. See Keywords

    Ramp link

    (Slang) Cards or strategies that accelerate resource development.

    S link

    Saboteur link

    A keyword that allows a unit to ignore Sentinel and defeat Shield tokens. See Keywords

    Sentinel link

    A keyword that protects other friendly units and the base. See Keywords

    Shielded link

    A keyword that gives a unit a Shield token when played or deployed. See Keywords (Related: Shield Token)

    Shield Token link

    A token upgrade that prevents damage to the attached unit. See Card Types

    Smuggle link

    A keyword that allows a card to be played from the resource zone. See Keywords

    Space Arena link

    The zone where space units are played. See Game Zones (Related: Arena, Ground Arena)

    Splash link

    (Slang) Including a small number of off-aspect cards in a deck for their powerful effects.

    Swing link

    (Slang) A significant change in the game state, often due to a powerful play or attack.

    T link

    Tech link

    (Slang) Specific card choices made to counter expected strategies in the meta.

    Token link

    A card type created by other cards, not included in decks. See Card Types

    Trait link

    A category on a card that may be referenced by other cards. See Card Types

    Trigger link

    The condition that causes an ability to resolve. See Abilities and Effects

    Tempo link

    (Slang) The pace and momentum of the game, often referring to who has the initiative in dictating play.

    U link

    Unit link

    A card type representing characters or vehicles that can attack and defend. See Card Types

    Unique link

    A card with a symbol before its name, limited to one copy in play per player. See Game Concepts

    Upgrade link

    A card type that attaches to and improves units. See Card Types (Related: Attach)

    V link

    Value Trade link

    (Slang) An exchange of cards or game actions that leaves you at an advantage.

    Vanilla link

    (Slang) A unit with no special abilities, just basic stats.

    W link

    When Defeated link

    A triggered ability that resolves when a card is defeated. See Abilities and Effects

    When Played link

    A triggered ability that resolves when a card is played. See Abilities and Effects

    Wipe link

    (Slang) A card or effect that removes all or most units from play.

    For more detailed information on specific topics, please refer to the linked sections in the Comprehensive Rules.
